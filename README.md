# Preprocessing-and-Data-analysis-Using-SQL

# Tech Layoffs SQL Data Analysis

## Project Overview
This project analyzes global tech layoffs using SQL.  
The dataset was cleaned and transformed before performing exploratory analysis.

## Dataset
Tech Layoffs dataset (2020–2023)

## Steps Performed

### 1. Data Cleaning
- Removed duplicate records using ROW_NUMBER
- Standardized company and industry names
- Converted date column into DATE format
- Handled missing values
- Removed unnecessary helper columns

### 2. Exploratory Data Analysis
- Identified top companies with the highest layoffs
- Analyzed layoffs by industry, country, and startup stage
- Examined yearly and monthly layoff trends
- Ranked companies with highest layoffs each year
- Calculated rolling layoffs trend
- Analyzed layoffs by funding categories

## SQL Skills Used
- CTE (Common Table Expressions)
- Window Functions
- DENSE_RANK
- Aggregations (SUM, AVG)
- CASE statements
- Data cleaning techniques

## Tools Used
- MySQL
